Hướng dẫn cài đặt và kích hoạt AutoCAD Architecture 2019

  1. Chạy file sfx.
  2. Chạy file Setup ở folder vừa giải nén từ file sfx ra.
  3. Chạy xong thì chúng ta tắt mạng đi.
  4. Mở phần mềm AutoCAD Architecture 2019 lên.
  5. Chọn Enter a Serial Number, và chọn Active nhập key ở trên rồi nhấn Next.
  6. Mở file trong folder xf-adsk2019 lên ( Lựa chọn bản 32bit hoạt 64bit phù hợp cho máy bạn nhé!)
  7. Copy Request Code vào rồi nhấn Generate rồi sau đó Copy Activation Key và dán vào phần mềm.
  8. Mở mạng lên vào phần mềm xài bình thường.
