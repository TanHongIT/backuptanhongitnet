Adobe Photoshop CC 2019 đã có nhiều thay đổi lớn, phần mềm sẽ không còn được hỗ trợ và sử dụng trên hệ điều hành Windows 32 – bit nữa, ngoài ra với những tính năng như phiên bản trước thì theo như cảm nhận và so sánh của mình thì phiên bản 2019 này khởi động nhanh hơn, thao tác mượt mà hơn, tốc độ xuất file tốt hơn rất nhiều.

Các tính năng mới trong phiên bản Photoshop 2019 v20.0.2 này:

  • Công cụ mới: Frame Tool thông minh
  • Cải tiến mạnh mẽ chức năng Content-Aware Fill
  • Chế độ Undo liên tiếp nhiều bước trước đó
  • Điểm tham chiếu (Reference Point) sẽ được ẩn đi
  • Double-click để chỉnh sửa text
  • Tự động hoàn tất tác vụ (Auto-commit)
  • Thu phóng giữ nguyên tỉ lệ của đối tượng
  • Ngăn việc thay đổi giao diện làm việc với Lock Workspace
  • Xem trước các chế độ hòa trộn (blend mode) trực tiếp
  • Chế độ đối xứng (Symmetry mode)
  • Chọn màu sắc với Color Wheel dễ dàng hơn
  • Nút quay lại màn hình Home
  • Cải thiện việc tự học ngay trong PTS
  • Tùy chọn tăng kích thước hiển thị giao diện người dùng
  • Match Font hỗ trợ các font tiếng Nhật
  • Cho phép gõ công thức toán học
  • Thay đổi cách hiển thị tên các layer có độ dài lớn
  • Hiển thị đoạn text mẫu Lorem Ipsum khi sử dụng Type tool

Cấu hình tối thiểu cho Adobe Photoshop CC 2019

  • Hệ điều hành: Windows 7 SP1 / 8 / 8.1 / 10 Chỉ 64-bit
  • Bộ vi xử lý đa lõi 2 GHz
  • RAM : tối thiểu 2 GB RAM (Mức đề nghị là 8GB còn 2gb dùng đợi chờ chắc chết)
  • Dung lượng ổ cứng trống: 5 GB
  • Màn hình hiển thị: tối thiểu 1024 x 768 (Mức đề nghị là 1280 x 800)
  • Hệ thống OpenGL 2.0 capable
  • VRAM: 512 MB

Hướng dẫn cài đặt và kích hoạt AutoCAD Architecture 2019

  1. Chạy file sfx.
  2. Chạy file Setup ở folder vừa giải nén từ file sfx ra.
  3. Chạy xong thì chúng ta tắt mạng đi.
  4. Mở phần mềm AutoCAD Architecture 2019 lên.
  5. Chọn Enter a Serial Number, và chọn Active nhập key ở trên rồi nhấn Next.
  6. Mở file trong folder xf-adsk2019 lên ( Lựa chọn bản 32bit hoạt 64bit phù hợp cho máy bạn nhé!)
  7. Copy Request Code vào rồi nhấn Generate rồi sau đó Copy Activation Key và dán vào phần mềm.
  8. Mở mạng lên vào phần mềm xài bình thường.
