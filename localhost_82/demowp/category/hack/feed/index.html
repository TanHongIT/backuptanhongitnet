Hiệu ứng của bộ code này tạo nên một phong cách rất lạnh lùng bí ẩn giống các Hacker trong phim. Bạn có thể sử dụng bộ source code này để tìm hiểu và học thêm kiến thức.

Ngoài ra các bạn hãy tải về và chỉnh sửa lại theo ý thích của các bạn rồi up lên host chia sẻ cho bạn bè mình cùng xem nhé!

Hướng dẫn:

  1. Các bạn tải về rồi giải nén ra.
  2. Để chỉnh sửa theo ý của các bạn thì các bạn hãy chỉnh sửa ở file index.html nhé! (dùng trình chỉnh sửa notepad++, hoặc những trình chỉnh sửa code khác.)
  3. Thay đổi nội dung trong các file đó đúng theo ý thích của bạn…
  4. Khi đã chỉnh xong thì bạn hãy up lên host để chạy. (có thể dùng host free như byethost hay 000webhost,… ).
  5. Gửi cho mọi người cùng xem thôi.

Nếu có vấn đề hay thắc mắc gì hoặc bạn không biết cách chỉnh sửa thì hãy bình luận bên dưới để cùng trao đổi và giúp đỡ các bạn nhé!
