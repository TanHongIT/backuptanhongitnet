Giới Thiệu Windows ME

Windows Millennium Edition (nội bộ Windows 4.90) là sản phẩm cuối cùng trong dòng sản phẩm Windows dựa trên DOS. Giống như Windows 95 và Windows 98, nó đã sử dụng cùng một nhân 16 bit bit lai chạy trên phiên bản đi kèm của DOS (MS-DOS 8.0).

Windows ME thường bị chỉ trích vì tính ổn định kém. Microsoft đã thực hiện nhiều thay đổi nội bộ ở cấp trình điều khiển thiết bị. Tuy nhiên, với thông tin rằng kiến ​​trúc này đã kết thúc, các nhà cung cấp đã không cập nhật trình điều khiển Windows 98 cho Windows ME.

Mặc dù Windows ME vẫn chạy trên DOS, giống như các phiên bản Windows trở lại 1.01, nó làm tê liệt khả năng khởi động hoặc thoát khỏi DOS. Nhiều người dùng Windows 95 và 98 đã dựa vào đó, vì nhiều trò chơi và tiện ích phổ biến vẫn dựa trên DOS. Nếu bạn muốn thiết lập một máy tính cũ hơn để sử dụng với các chương trình DOS, chúng tôi khuyên bạn không nên sử dụng gì muộn hơn Windows 98 SE.

Thông minh về tính năng, Windows ME thay đổi giao diện máy tính để bàn ngang với Windows 2000. Nó bổ sung Windows Movie Maker, Internet Games, System Restore, Explorer Web View, Menu cá nhân hóa và các hộp thoại chung mới.

Giới Thiệu Windows Whistler

Windows_Whistler là tên mã trước khi phát hành cho Windows XP, hoặc NT 5.1. Nó là sự kế thừa cho Windows 2000. XP là phiên bản đầu tiên không cạnh tranh với phiên bản Windows dựa trên DOS, cuối cùng đã hợp nhất dòng Windows thành một sản phẩm 32 bit đơn thuần.

Sự phát triển của Windows XP bắt đầu vào ngày 5 tháng 2 năm 1999 dưới dạng Windows Neptune . Neptune sẽ là kế thừa của Windows ME , mặc dù dựa trên nhân NT . Microsoft đã hợp nhất các nhóm làm việc trên Neptune với Windows Odyssey , người kế nhiệm của Windows 2000 , vào đầu năm 2000. Dự án kết quả, có tên mã là “Whistler“, đã trở thành Windows XP.
