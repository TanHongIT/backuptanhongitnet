Chỉnh sửa file hosts có thể khiến bạn không thể truy cập Internet nếu file không được sửa đổi chính xác. Phần mềm này sẽ hướng dẫn các bạn chỉnh sửa file hosts trong hệ điều hành Windows.

Vì sao lại có file Host ???

Khi bạn nhập một tên miền trong trình duyệt mà bạn muốn kết nối, trước tiên, ứng dụng của bạn phải chuyển đổi nó thành địa chỉ IP mà nó sẽ sử dụng để kết nối. Cách thức các tên máy chủ này được giải quyết theo địa chỉ IP được ánh xạ của chúng được gọi là Domain Name Resolution.

Trên hầu hết tất cả các hệ điều hành cho dù chúng là Apple, Linux, Unix, Netware hay Windows, phần lớn các độ phân giải từ tên miền đến địa chỉ IP đều được thực hiện thông qua một thủ tục gọi là DNS.

DNS là viết tắt của Hệ thống tên miền và là dịch vụ phân giải tên miền tiêu chuẩn được sử dụng trên Internet, Có một cách khác để giải quyết tên miền mà không cần sử dụng Hệ thống tên miền, đó là sử dụng tệp HOSTS của bạn.

Hầu như mọi hệ điều hành giao tiếp qua TCP / IP, tiêu chuẩn giao tiếp trên Internet, đều có một tệp gọi là tệp HOSTS. Tệp này cho phép bạn tạo ánh xạ giữa tên miền và địa chỉ IP Một cách khác để nghĩ về tệp máy chủ là trình dịch và chuyển hướng địa chỉ.

Vì sao bạn cần chỉnh sửa File Host ???

Đa số sửa file host để vào facebook, chặn phần mềm update, chặn website check key bản quyền phần mềm là chính. Tùy theo nhu cầu và mục đích của mình mà các bạn cần chỉnh sửa file hosts đó cho phù hợp.

Và khi chỉnh sửa file hosts có thể khiến bạn không thể truy cập Internet nếu file không được sửa đổi chính xác.
