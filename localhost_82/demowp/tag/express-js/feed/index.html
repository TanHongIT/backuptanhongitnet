]]> MEAN Stack là một từ viết tắt của MongoDB, Express, Angular và Node.js – nhờ đó tất cả bốn phần này được tích hợp để tạo thành các ứng dụng JavaScript full-stack được giải pháp.
 
Hầu như mọi người phát triển web trên thị trường đang cố gắng trở thành nhà phát triển ứng dụng ngăn xếp MEAN.


 Bạn sẽ học cách:

  • Tạo máy chủ Nodejs với Express.js
  • Sử dụng ES6 với Nodej
  • Tạo APIS an toàn với Nodej, Express và Mongoose
  • Bảo mật APIS an toàn với  Hộ chiếu và JWT
  • Xác thực xã hội ( Twitter, Google, Github ) trong Nodej
  • CRUD ( Tạo, Đọc, Cập nhật, Xóa ) Hoạt động trong Góc
  • Xây dựng ứng dụng góc với vật liệu góc, bảng dữ liệu vật liệu và hộp thoại tài liệu
  • Tìm hiểu cách sử dụng các thành phần vật liệu góc trong ứng dụng thế giới thực
  • Xác thực mã thông báo web JSON trong góc
  • Xác thực xã hội ( Twitter, Google, Github ) trong Angular
  • Tạo tệp PDF trong Nodej
  • Tải xuống tệp PDF trong Angular
  • Document APFUL APIS với Swagger
  • Gửi email trong Nodejs

Đối tượng, mục tiêu:

  • Nhà phát triển góc trung gian muốn tìm hiểu về phát triển web đầy đủ
  • Nhà phát triển Nodejs muốn tìm hiểu về phát triển web đầy đủ
 

Những gì bạn học được:

  • Tích hợp bất kỳ ứng dụng góc nào với Nodejs
  • CURD (Tạo, Đọc, Cập nhật, Xóa) Thao tác với Nodejs, Express và Mongoose
  • CURD (Tạo, Đọc, Cập nhật, Xóa) Thao tác với Nodejs, Express và Mongoose
  • Xác thực mã thông báo web JSON với PassportJs trong MEAN Stack
  • Xác thực xã hội (Twitter, Github, Google) với PassportJs trong MEAN Stack
  • Gửi email trong Nodejs
 

Yêu cầu: 

  • NodeJS cơ bản / Express và kinh nghiệm góc là bắt buộc
  • Kiến thức cơ bản về HTML, CSS và Javascript là bắt buộc
