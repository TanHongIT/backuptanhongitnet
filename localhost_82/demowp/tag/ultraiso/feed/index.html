UltraISO 

Phần mềm được biết đến là phần mềm ghi đĩa ra đời sớm và chất lượng được đánh giá cao. Với giao diện cổ điển, các tính năng được sắp xếp hợp lý, bao gồm các công cụ tạo, chỉnh sửa, trích xuất ảnh đĩa và quản lý cấu trúc của chúng. Quá trình tạo ảnh đĩa diễn ra nhanh chóng và thông báo cho bạn biết khi tiến trình hoàn tất.

Những tính năng, lợi ích của phần mềm này:

  • Mở file ISO.
  • Chỉnh sửa trực tiếp file ISO.
  • Thêm, sửa, xóa tập tin cũng như thay đổi tên file, folder từ file ISO nguồn.
  • Tạo file ISO dữ liệu, ghi file ISO Boot cài Win.
  • Tạo ổ đĩa ảo, hỗ trợ duy nhất 1 đĩa ảo trên máy tính Windows.
  • Hỗ trợ hầu hết định dạng file ảnh, phổ biến như: .ISO, .BIN, .IMG, .CIF, .NRG, .BWI.
  • Hỗ trợ theo chuẩn ISO 9660 Level1/2/3/
  • Hỗ trợ mọi hệ điều hành Windows XP/7/8/8.1/10, kể cả Windows Server.
  • Trích xuất tập tin, thư mục từ file ISO.
  • Thiết lập ẩn, hiện dữ liệu trong file ảnh.
  • Giao diện thân thiện, dễ sử dụng kể cả đối với người dùng lần đầu biết tới.
  • Hỗ trợ tích hợp các loại tài liệu vỏ, mở tệp hình ảnh thông qua nhấp đúp chuột vào chúng hoặc trình đơn Nhấp chuột phải của con chuột với hộp thoại mở.
  • Giao diện cửa sổ đôi là rất thuận tiện để sử dụng.
  • Có nhiều công cụ bổ sung, bạn có thể tạo đĩa CD khởi động ALL in 1, quản lý tập tin ảnh đĩa CD, thậm chí ổ CD / DVD ảo, những chức năng này thực sự mạnh mẽ.
  • Hệ điều hành được hỗ trợ: Windows XP, Server 2003, Vista, Server 2008, Windows 7, Server 2012, Windows 8, Windows 8.1, Windows 10 (32-bit, 64-bit).
