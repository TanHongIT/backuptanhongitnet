Facebook – Mạng xã hội phổ biến nhất hành tinh hiện nay đã bắt đầu cung cấp cho người dùng những biểu tượng cảm xúc  để chia sẻ cảm giác như thế nào ở mỗi bài viết được đăng tải .

 
 
 
Các biểu tượng cảm xúc gồm : Like (thích), Love(Yêu thích), Haha(Cười) , Sad( buồn), Angry (tức giận).
Và Hôm nay mình sẽ hướng dẫn các bạn cách thay đổi biểu tượng Angry(tức giận) thành biểu tượng ” Dislike” nhé.
Đầu tiên bạn tải tiện ích sau về trình duyệt mà bạn đang dùng nha: Tampermonkey.
 link trình duyệt Opera : https://goo.gl/CLAVFb
 link trình duyệt Edge :  https://goo.gl/QmfU1D
 link trình duyệt firefox : https://goo.gl/zpwX5Z
 link trình duyệt Chrome: https://goo.gl/ttbWLA
 
Ở bài viết này mình sẽ làm trên  Chrome , các trình duyệt khác các bạn cứ tải rồi làm tương tự nhé.
 
Sau khi nhấn vào link phù hợp với trình duyệt đang dùng thì bạn nhấn Thêm vào chrome ( Add To Chrome) , sau đó nhấnVào Thêm Tiện Ích .
 
 
Bạn chờ nó add vào trình duyệt nữa thôi, biểu tượng của tiện ích trên trình duyệt giống ở hình sau:
Vậy là xong bước đầu . Bây giờ bạn hãy nhấn vào link bên dưới rồi sau đó nhấn vào nút Install nhé.
 
link ở đây : https://goo.gl/y1u9z9
 
 
Bạn chỉ cần nhấn vào Install là xong mọi thứ rồi nhé.
 
bây giờ bạn vào trang chủ facebook và kiểm ra thử thôi.
                        https://www.facebook.com
 
Nếu như bạn thấy có biểu tượng như hình dưới thì bạn đã đổi thành công rồi nhé.
Nếu bạn vẫn chưa thấy thay đổi thì hãy nhấn F5 để reload lại trang facebook và thay đổi nhé.
 
 
Chúc các bạn thành công.
