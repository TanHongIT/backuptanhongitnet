Đối tượng, mục tiêu:

  • Khóa học Grunt.js này dành cho những người quen thuộc với HTML, CSS và Javascript. Không có kiến ​​thức trước về Grunt.js / Node.js / npm là cần thiết. Tất cả mọi thứ được mô tả từ đầu.

Những gì bạn học được:

  • cài đặt và cấu hình Grunt.js, npm, Node.js
  • rút gọn và hợp nhất các tệp CSS và JS -> sẽ cải thiện tốc độ tải trang web
  • tự động thêm tiền tố của nhà cung cấp, sau khi thiết lập dự án, bạn sẽ không cần phải suy nghĩ về tiền tố của nhà cung cấp nữa
  • tìm lỗi Javascript gần như ngay lập tức -> không mất nhiều thời gian để gỡ lỗi mã Javascript
  • giảm đáng kể thời gian cần thiết để chạy nhiều tác vụ cùng một lúc

Yêu cầu:

  • html, css, javascript
