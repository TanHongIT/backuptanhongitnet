]]> Easy Context Menu – Menu windows chuột phải hay còn gọi lại Context Menu là phần mềm thêm vào menu chuột phải danh sách các chức năng khi bạn click chuột phải vào màn hình Desktop.

Với Easy Context Menu (ECM) cho phép bạn thêm một loạt các lệnh, các phần mềm, các tinh chỉnh hữu ích và cho Desktop, My Computer, ổ đĩa, File,… vào menu chuột phải.

Việc thêm phần mềm vào Context Menu sẽ giúp bạn tiết kiệm thời gian hơn khi mở phần mềm thường sử dụng. Chỉ cần một cái click chuột là gọi nhanh phần mềm. Nó giúp truy cập hầu hết các thành phần Windows được sử dụng một cách nhanh chóng và dễ dàng.

Các mục chức năng chính:

  • Desktop Menu
  • My Computer Menu
  • Drives Menu
  • Folder Menu
  • File Menu
  • System tools sub menu
  • Turn Off Computer sub menu
  • Exe file context menu

Hướng dẫn sử dụng Easy Context Menu

Đầu tiên bạn tải về giải nén ra. Chạy phần mềm với quyền Administrator.

Lúc mở lên là tiếng anh chúng ta chọn Options => Languages chỉnh thành tiếng việt.

Cách Thêm Phần mềm vào Menu chuột phải trong windows

Khi vào giao diện chính của Easy Context Menu bạn hãy click chọn vào các ô vuông tương ứng theo tiện ích bạn mún bỏ vào menu chuột phải. Sau đó bạn nhất vào nút Apply Change(Biểu tượng con chuột có dấu cộng).

Bên trong Easy Context Menu cũng có tích hợp một số công cụ tiện ích của hãng Sordum như: Delete Temp Files, Restart Windows Explorer, Show or Hide Hidden Files/Folders, Show or Hide File Extensions, Rebuild Shell Icon Cache, Select All Files, Copy IP address, Reduce Memory, Fix Print Spooler, Take Ownership… Bạn có thể dễ dàng thêm chúng vào Menu Chuột phải để sử dụng.

Và đây là kết quả: 

Với tùy chọn Enable Shift – Click, sẽ hiện thị khi bạn giữ phím Shift và Click phải chuột để hiện lên một Menu khác.

Thêm, di chuyển các phần mềm, tiện ích vào Menu Context

Chúng ta chọn menu trên File(Tập tin) => List Editor(Sửa danh sách).

Ấn Add New(thêm mới) xong chọn đường dẫn đến File. Sau đó bạn lưu lại bạn sẽ thấy nó xuất hiện trong ECM.

Hoặc bạn có thể di chuyển các tiện ích qua lại giữa các Target Context nữa.

Cách detele làm sạch Menu Nhấp chuột phải

Có một số phần mềm tự Add phần mềm của mình vào Menu Chuột phải mà không xin phép. Bạn có thể sử dụng tính năng dọn dẹp của ECM để xóa tất cả các mục không mong muốn. Vào Menu File –> Context Menu Cleaner

Chỉ cần chọn một mục không mong muốn và nhấp vào nút Delete(Xóa) để xóa nó khỏi menu ngữ cảnh của bạn là xong.

Sửa Lỗi Trên Windows 10

Nếu trên Windows 10 bạn không thể Thêm Phần mềm vào Menu, thì Check theo hình bên dưới để sửa lỗi.
