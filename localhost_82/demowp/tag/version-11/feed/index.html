Những tính năng hữu ích của ESET Internet Security

  • Computer Protection: Giúp bạn bật tính năng bảo vệ theo thời gian thực.
  • Internet Protection: Giúp bạn tùy chỉnh các tính năng bảo mật trên internet bao gồm web, email, chống spam và lừa đảo trực tuyến.
  • Network Protection: Giúp bạn tùy chỉnh các bảo mật về an ninh mạng như tường lửa cá nhân và tấn công mạng có chủ đích.
  • Bảo vệ thanh toán trực tuyến.
  • Các tùy chỉnh nâng cao khác.

Yêu cầu hệ thống:

  • Bộ vi xử lý hỗ trợ: Intel® hoặc AMD x86-x64.
  • Hệ điều hành: Microsoft® Windows® 10, 8.1, 8, 7, Vista.
  • Hỗ trợ cả 2 phiên bản 32bit và 64bit.
